## 🔍 Detailed Notebook Breakdown

### 1. 🚀 [First_Day.ipynb](file:///c:/Users/harsh/OneDrive/Desktop/Claude_Project/Offline_Students/First_Day.ipynb)

## 🛠️ Getting Started & Prerequisites

To run these notebooks, you will need a Python environment setup on your system.

### 📋 Prerequisites
1. **Python 3.x** installed.
2. **Jupyter Notebook** or **JupyterLab** or **VS Code (with Jupyter Extension)**.
3. Install required libraries used in some of the advanced notebooks:
   ```bash
   pip install numpy sympy
   ```

### 💻 Running the Notebooks
1. Clone or open this folder on your machine.
2. Launch Jupyter Notebook from your terminal:
   ```bash
   jupyter notebook
   ```
3. Open any `.ipynb` file from the list, run cells sequentially using `Shift + Enter`, and input values in the interactive prompts as required!
